# PHP Video Platform
> PHP ile geliştirilmiş YouTube benzeri video paylaşım platformu.

## Açıklama
Bu proje, PHP ile geliştirilmiş YouTube benzeri bir video paylaşım platformudur. Kullanıcıların video yükleyip izleyebildiği temel bir içerik sistemi üzerine kuruludur.

Proje başlangıçta custom MVC yapısı ile geliştirilmiştir. Geliştirme sürecinde mimari ihtiyaçların artması nedeniyle şu anda kendi geliştirdiğim lightweight PHP framework kullanılarak yeniden yapılandırılma sürecindedir.

## Kullanılan Teknolojiler

- PHP
- MySQL
- Redis
- HTML / CSS / JS
- Custom Lightweight PHP Framework

## Çalıştırma
```bash
docker compose up -d
```
